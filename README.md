# Real-Time Character Counter

## 📌 Project Overview
This project is a simple web application that provides a **real-time character counter** for a text area. As the user types, the application continuously updates the number of characters entered and displays how many characters remain based on a predefined maximum limit.

The project demonstrates the use of **HTML**, **CSS**, and **JavaScript** to create an interactive and user-friendly text input experience.

## ✨ Features
- Text area for users to type a message.
- Displays the maximum character limit (e.g., **200 characters max**).
- Updates the character count instantly as the user types.
- Shows the current character count in the format:
  - **150/200 characters**
- Displays the number of remaining characters.
- Prevents users from typing beyond the maximum limit.
- Displays a warning message when the character limit is reached.

## 🛠️ Technologies Used

- JavaScript (DOM Manipulation & Event Listeners)

## 🚀 How It Works
1. The user types a message into the text area.
2. JavaScript listens for the `input` event.
3. The application counts the number of characters entered.
4. The character counter updates in real time.
5. If the maximum limit is reached, no additional characters can be entered.
6. A warning message is displayed to inform the user that the character limit has been exceeded.
